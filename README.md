# truffle-unbox-react-hooks

A starter kit for Dapps using Truffle and React Hooks.

To install, type "yarn" at /client.

To run with fixed ganache keys + truffle console:
```
./rganache
```
In another console:
```
./tconsole

migrate --reset
```
In another console:
```
cd client
npm start
```


Or just as general develop env:
To run:
```
truffle develop
migrate --reset
```

In another console:
```
cd client
npm start
```
